# API

## Development

### Install

> pip install yml-api

### Create Project

> mkdir $project_name
> 
> cd $project_name
> 
> python -m api
> 
> python manage.py sync

### Run Project

> python manage.py runserver

### Test Project

> python manage.py test

## Docker

### Image Creation

> python -m api build

### Run Project

> python -m api up

### Project Log

> python -m api log

### Stop Log

> python -m api down

### Test Project

> python -m api test
